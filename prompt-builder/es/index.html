Personajes Instrucciones §5 requerido

Un bloque por personaje. El ESTADO DEL PERSONAJE se establece aquí — el modelo hace seguimiento desde estas bases.

Rastreadores Visibles Instrucciones §6 visible
Mostrado en cada respuesta. Valores mantenidos en contexto por el modelo — sin backend. Se reinicia al finalizar el chat.
Rastreadores Ocultos Instrucciones §6 oculto
Nunca se muestran. Gobiernan el comportamiento solo mediante reglas de umbral. Valores mantenidos en contexto.
Rastreadores visibles — modo inmersivo
Mismos umbrales y reglas de comportamiento que las versiones visibles. Valor rastreado internamente — nunca se muestra. No combinar con la versión visible del mismo rastreador.
Rastreadores Híbridos Instrucciones §6 híbrido
Rastreado en silencio. La etiqueta de etapa aparece una vez al cambiar, luego vuelve al rastreo invisible.
Sistema de Memoria Instrucciones §7 requerido

El complemento de Memoria Persistente. Rige CORE MEMORY, CHARACTER STATE y EVENT MEMORY. EVENT MEMORY comienza vacía y se construye en contexto solo para esta sesión.

Siempre incluido en cada exportación — sin configuración necesaria.
Motor de Salto Temporal Instrucciones §8 requerido

Gestiona el ritmo y la progresión temporal. Actualiza CORE MEMORY y CHARACTER STATE cuando se activa. Funciona mejor junto al rastreador de visualización de tiempo / fecha / ubicación.

Siempre incluido en cada exportación — sin configuración necesaria.
